Static Demo UI

Query CVEs and review one-page analysis.

This is a standalone HTML version of the UI for localhost or GitHub Pages. It uses bundled sample data so it works on a static host without a backend.

For live NVD, EPSS, and KEV API ingestion, use the Next.js app. Static hosting can render the interface, but the production aggregation flow still needs a server.
